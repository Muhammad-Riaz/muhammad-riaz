    Shining Stars Promotors

    It’s a web based application, which is used to store and manage data for a system which promotes Amazon products arround the world.

    Role

    • Lead UI/UX Designer
    • Lead Frontend Developer

    Tools & Technologies Used

    • Adobe Photoshop, Illustrator & XD
    • Html, CSS, bootstrap & JavaScript

    Type

    • Responsive Web Application

    Overview

    Shining Stars Promotors Web App is an application where a user (admin) stores detailed lists of products. The products lists are accessible to the other users called agents. Agents sell the products to Amazon users and get their reviews on the products. Agents store orders and reviews’ detail in the application, which is used for confirmation. After the completion of review the seller pay commission to the agents. The application also has a chatting module where agents can discuss problems regarding product, order, review, commission etc. with admin.

    I chose the project because I worked as an agent before with Amazon product promoters. I had an idea of the project. So, a person related to a field can understand and solve problems better than an outsider.

    Project Timeline

    • January - July 2021

    Problem

    As admins were using different applications like Google Form , Google Sheets and a third-party web application for storing the images, which was very complicated, unsecure and hard to use. There was no chatting module for users. They had to use WhatsApp or Messenger for discussions.

    Solution

    Shining Stars Promotors provides an easy solution to both the admin and agents. User can easily perform tasks like uploading products through just a file, searching, chatting, storing, automatic commission calculation, data charts, storing orders and reviews’ data.

    User Research

    As the app is related to both admin and agents so I had to get the input from both users for that I conducted a survey from both type of users.

    In Admin POV survey questions were related to managing products, agents, commissions, security and response to agent complaints.

    In Agent POV survey questions were about use of other applications, complexity of use, time, sharing personal information.

    Insights

    From the survey I gathered the following data. Both admin and agents had problems and concerns with the already available system.
